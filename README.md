sentiment_analysis

My goal is to analyze how different musical structures convey human emotions and cultural context. Specifically, this project aims to build a computational model that listens to traditional Korean percussion (Samulnori) audio, automatically recognizes the specific rhythmic patterns (Jangdan), and interprets their historical and cognitive origins.

Tech/skill
import streamlit as st loads the library that easily turns your Python code into a web app.
librosa is the industry-standard library for music and audio analysis. It is used to extract rhythmic patterns from Samulnori audio, while librosa.display explicitly loads the visualization module to plot acoustic waveforms.
matplotlib.pylot used for data visualization.
