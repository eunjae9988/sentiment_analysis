sentiment_analysis

My goal is to analyze how different musical structures convey human emotions and cultural context. Specifically, this project aims to build a computational model that listens to traditional Korean percussion (Samulnori) audio, automatically recognizes the specific rhythmic patterns (Jangdan), and interprets their historical and cognitive origins.
